Data Layout


The ' /data ' URL give you access to get and set data on the sensor-level.

A brief breakdown of each layout is as follows:

Getting Data

To get data from the API service, simply call a GET request to the respective URL for the sensor you're looking for.

GET " eel5632.tylersmith.us/data/YOUR_SENSOR_ID_HERE "

This can be done by simply entering the above URL in a browser window. This should return a JSON object with relevant sensor information, if the sensor exists. Otherwise, it will return a blank JSON object.
